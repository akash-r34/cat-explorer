# 🐾 CatExplorer: User Guide & Feature Walkthrough

Welcome to **CatExplorer**, your cinematic dashboard for managing feline diagnostic registries. This guide will walk you through the key features and workflows of the application.

## 📊 Dashboard Insights
Upon logging in, you are greeted with the **Analytics Dashboard**. This view provides a high-level overview of your entire cat registry.

- **KPI Cards**: Instantly see total cat counts and key metrics.
- **Dynamic Charts**: Visualize data distribution across your registry.
- **Quick Links**: Jump directly to common actions.

![Dashboard](assets/dashboard.png)

## 🐈 Managing the Cat Registry
The **Cat Explorer** is your main hub for all record management.

### Finding Cats
Use the **Live Search** bar to instantly filter through thousands of records. The list updates in real-time as you type, ensuring you find the right diagnostic profile in seconds.

![Explorer](assets/explorer.png)

### Adding & Editing Cat Profiles
Managing records is simple and secure. When adding or editing a cat, the system performs **Real-time Validations** to ensure high data quality:

- **Name**: Must be between 2-50 characters and contain only letters/spaces.
- **Age**: Must be a whole number between 0 and 30.
- **Description**: Supports detailed diagnostic info up to 500 characters.

If you enter invalid data, helpful error messages will appear instantly to guide you.

![Details](assets/details.png)

## 📥 Data Export
Need to share your registry data? Use the **Export CSV** feature located in the Cat Explorer view.

- **Professional Format**: Generates a clean CSV file with Name, Age, and Description.
- **Compatibility**: Optimized for seamless opening in Excel, Google Sheets, or Numbers.
